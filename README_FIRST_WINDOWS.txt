OPTIMUS 7.0.1 - LANDON MOTOR WORKS WINDOWS QUICK START

1. Extract the complete ZIP to a normal folder such as C:\Optimus-7.
   Do not run the files from inside the ZIP preview.

2. Double-click WINDOWS_SETUP.bat one time.
   Enter the OpenAI API key when prompted.

3. Double-click local.bat each time you want to start Optimus.

4. The official command center opens automatically at:
   http://127.0.0.1:8000

5. The local access token is loaded automatically into that browser tab.
   You normally do not need to paste it manually.

6. Press Ctrl+C in the black local.bat window to stop Optimus.

Use CHECK_OPTIMUS.bat to test configuration, OpenAI access, and the code suite.
Use RESET_OPENAI_KEY.bat when replacing an invalid or revoked API key.
Use DIAGNOSE_ESTIMATOR.bat for one live labor-and-parts estimator test.
