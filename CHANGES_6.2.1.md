# Optimus 6.2.1 changes

- Added `WINDOWS_SETUP.bat` for double-click installation.
- Added `local.bat` and `RUN_OPTIMUS_LOCAL.bat` for local startup.
- Added `CHECK_OPTIMUS.bat` for API and automated validation.
- Added secure PowerShell API-key entry and automatic access-token generation.
- Added automatic browser opening after the local health endpoint is ready.
- Added clipboard transfer of the local Optimus access token.
- Added browser session storage for the app access token.
- Added strict startup rejection for placeholder credentials.
- Added Windows distribution tests.
